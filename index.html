Google Chat Enhancerプライバシーポリシー

最終更新日: 2026年7月9日

English version below ↓

Google Chat Enhancer（以下「本拡張機能」）は、Google ChatにEnterで改行、下書き一覧、送信履歴などの補助機能を追加するChrome拡張機能です。本ポリシーは、本拡張機能が取り扱うデータとその取り扱い方法を説明します。

基本方針

  • 本拡張機能は、開発者独自のサーバーを一切持ちません。開発者がユーザーのデータを受信・保存・閲覧することはありません。
  • 本拡張機能は、Google Chat（https://chat.google.com/*）上でのみ動作します。
  • 解析（アナリティクス）、トラッキング、広告を目的としたデータ収集は一切行いません。
  • 本拡張機能は、ユーザーデータを販売・転送したり、信用力の判断や融資目的に使用したりしません。

取り扱うデータ

本拡張機能は、機能を提供するために以下のデータを処理・保存することがあります。

  • 設定情報: Enterで改行、下書き一覧、送信履歴など各機能のオン／オフ設定。
  • 下書きデータ: Google Chatの入力欄に書いた下書き本文、会話またはスペース名、Google Chat内の移動先パス、保存日時、スレッドID。下書き本文は最大7,000文字まで保存されます（同期ストレージの容量に収めるため、より短く切り詰められる場合があります）。
  • 送信履歴データ: 本拡張機能が検知した最近の送信メッセージ本文、会話またはスペース名、Google Chat内の移動先パス、送信日時、スレッドID。送信履歴は最大100件、本文は各件最大500文字まで保存されます。

本拡張機能は、健康情報、財務情報、認証情報、位置情報、閲覧履歴、広告目的の行動情報を意図的に収集しません。ただし、ユーザーがGoogle Chatの下書きやメッセージ本文にこれらの情報を書いた場合、その内容が下書きデータまたは送信履歴データとして保存される可能性があります。

データの利用目的

取得したデータは、Google Chat上で下書きを保存・再開する、最近の送信先へ戻る、各機能の設定を維持するなど、本拡張機能の単一目的とユーザー向け機能を提供するためにのみ使用します。それ以外の目的には使用しません。

第三者への送信

  • 本拡張機能は、開発者、開発者が管理するサーバー、または任意の第三者サービスへユーザーデータを送信しません。
  • ユーザーがGoogle Chatでメッセージを送信した場合、その通信はGoogle Chat本来の機能としてGoogleにより処理されます。本拡張機能は、その通信経路に独自の送信先を追加しません。
  • chrome.storage.syncに保存されたデータは、ユーザーがChromeの同期機能を有効にしている場合、Google Chromeの仕組みによりユーザー自身のChrome環境間で同期されることがあります。この同期は開発者による受信・閲覧を意味しません。

データの保存

  • 設定情報および下書きデータは、Chromeのstorage.syncを用いて保存されます。
  • 送信履歴データは、Chromeのstorage.localを用いてユーザーの端末上に保存されます。
  • これらのデータが開発者や開発者のサーバーへ送信されることはありません。

データの保持期間と削除

  • 下書きデータは、ユーザーが該当メッセージを送信した場合、入力欄を空にした場合、または下書き一覧から削除した場合に削除されます。
  • 送信履歴データは、最大100件まで保持されます。ユーザーは送信履歴一覧から個別に削除できます。
  • 本拡張機能をChromeから削除する、またはChromeの拡張機能データを削除することでも、保存データを削除できます。

限定的使用

本拡張機能は、Chrome Web Storeのユーザーデータポリシー（Limited Use requirementsを含む）を遵守します。ユーザーデータは、本拡張機能の開示された機能を提供・改善するためにのみ使用し、広告、トラッキング、販売、目的外の転送、人による閲覧には使用しません。

Google Chat Enhancer — Privacy Policy

Last updated: July 9, 2026

↑ 日本語版は上にあります

Google Chat Enhancer (the "Extension") is a Chrome extension that adds helper features to Google Chat, including new line on Enter, a drafts list, and sent history. This policy explains what data the Extension handles and how.

Core principles

  • The Extension has no developer-operated server. The developer never receives, stores, or views your data.
  • The Extension runs only on Google Chat (https://chat.google.com/*).
  • The Extension performs no analytics, tracking, or advertising-related data collection.
  • The Extension does not sell or transfer user data, and does not use user data for creditworthiness or lending purposes.

Data the Extension handles

The Extension may process and store the following data to provide its features:

  • Settings: on/off settings for features such as new line on Enter, drafts list, and sent history.
  • Draft data: draft message text entered in Google Chat, conversation or space name, navigation path within Google Chat, saved timestamp, and thread ID. Draft message text is stored up to 7,000 characters (it may be truncated shorter to fit the sync-storage size limit).
  • Sent history data: recently sent message text detected by the Extension, conversation or space name, navigation path within Google Chat, sent timestamp, and thread ID. Sent history is stored up to 100 items, with message text up to 500 characters per item.

The Extension does not intentionally collect health information, financial information, authentication information, location information, browsing history, or behavioral data for advertising. However, if you type such information into a Google Chat draft or message, that content may be stored as draft data or sent history data.

How the data is used

The data is used only to provide the Extension's single purpose and user-facing features, such as saving and resuming drafts in Google Chat, returning to recent conversations, and preserving feature settings. It is not used for any other purpose.

Disclosure to third parties

  • The Extension does not transmit user data to the developer, a developer-operated server, or any arbitrary third-party service.
  • When you send a message in Google Chat, that communication is processed by Google as part of Google Chat's normal functionality. The Extension does not add any additional destination to that communication path.
  • Data stored in chrome.storage.sync may be synchronized across your own Chrome environments by Google Chrome if you have Chrome sync enabled. This does not mean the developer receives or views the data.

Data storage

  • Settings and draft data are stored using Chrome's storage.sync.
  • Sent history data is stored on your device using Chrome's storage.local.
  • None of this data is transmitted to the developer or a developer-operated server.

Data retention and deletion

  • Draft data is deleted when you send the corresponding message, clear the input field, or delete the draft from the drafts list.
  • Sent history data is kept up to 100 items. You can delete individual sent history items from the sent history list.
  • You can also delete stored data by removing the Extension from Chrome or clearing the Extension's data in Chrome.

Limited use

The Extension complies with the Chrome Web Store User Data Policy, including the Limited Use requirements. User data is used only to provide or improve the Extension's disclosed functionality, and is not used for advertising, tracking, sale, unrelated transfer, or human review.
